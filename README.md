# Footer Visual Window Organizer

Organize and identify your Chrome windows at a glance with visual footer indicators. Assign distinct colors to each window for easy organization and management.

## Features
* **Visual Window Markers:** Subtly change the color of each Chrome window for instant identification.
* **Easy Organization:** Manage multiple browser windows with visual clarity.
* **Simple Interface:** One-click color assignment per window.
* **Privacy First:** Runs 100% locally. No remote code, no analytics.

## Installation
1. Load the extension in Chrome's developer mode
2. Open `chrome://extensions/`
3. Enable "Developer mode" and load the unpacked extension folder

## Usage
Click the extension icon and select a color to assign to your current window. The footer will display your chosen color, making it easy to visually distinguish between multiple windows.
